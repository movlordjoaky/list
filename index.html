GOOGLE DRIVE API

Для синхронизации нужен OAuth Client ID.
Создаётся один раз бесплатно:

1. Открой console.cloud.google.com
2. Создай проект → APIs & Services → Credentials
3. Create Credentials → OAuth client ID → Web application
4. В Authorized JavaScript origins добавь адрес этой страницы
5. Скопируй Client ID и вставь ниже
